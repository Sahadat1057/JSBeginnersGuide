Concept: DOM (Document Object Model)

Explanation:

The Document Object Model (DOM) is a programming interface that acts as a bridge allowing JS to interact with an HTML page. The DOM isn't part of the JS language, but instead a web API that is built together using multiple other APIs. The DOM communicates with the browser using properties, methods and events. These methods include the querySelectorAll(), createElement(), getElementById(). These properties include innerHTML property, addEventListener() etc.
